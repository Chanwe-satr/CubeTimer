# CubeTimer

This folder keeps the original single-file program and a split static version for easier maintenance.

- `cubetimer.txt` is the untouched original source file.
- `index.html` is the browser entry point.
- `styles.css` contains the extracted styles.
- `app.js` contains the extracted timer and cube logic.

Open `index.html` directly in a browser to run the app.
